第一节 - 栈的数据结构

栈是一种遵循后进先出（LIFO - Last In First Out）的有序集合。新添加的或者待删除的元素都保存在栈的顶端，称之为栈顶。另一端叫栈底，旧的元素都靠近栈底。

ES6WeakMap实现

为什么要用ES6的WeakMap来实现，因为他可以确保属性是私有的，WeakMap可以存储键值对，其中键是对象，值可以是任意数据类型。使用闭包把Stack类包起来，确保items不被外部访问修改。
